### 变量与类型

#### 基本变量类型：

- 整型
- 浮点型
- 字符串型
- 布尔型
- 复数型

#### 类型检查：

```python
>>> a = 'heelo world'
>>> type(a)
<class 'str'>
```

#### 类型转换：

- `int()`：将一个数值或字符串转换为整数，可指定进制
- `float()`：将一个字符串转换为浮点型
- `str()`：将指定对象转换为字符串形式，可指定编码
- `char()`：将整数转换为该编码对应的字符串（一个字符）
- `ord()`：将字符串（一个字符）转换为对应的编码（整数）

#### 变量命名：

- 硬性规则：
  - 由字母（广义的unicode字符，不包括特殊字符）、数字、`_` 构成，数字不能开头
  - 大小写敏感
  - 不要跟关键字和系统保留字重复
- PEP8要求：
  - 小写字母编写，多个单词用 `_` 连接
  - 受保护的实例属性用 `_` 开头
  - 私有的实例属性用 `__` 开头
